## 叙事视角

填写人称、视角限制、信息透明度和旁白距离。

## 句式口味

记录句子长度、对白密度、描写颗粒度和节奏偏好。

## 节奏禁忌

记录不希望出现的拖沓方式、解释方式或文风偏差。

## Inject 使用

确认内容稳定后，可以按需要把 `inject.always` 改为 `true`，让写作 profile 默认带入。
